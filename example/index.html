Free Dialog 使用说明

简介

一款主要用于移动端浏览器的简单弹窗插件，默认提供气泡、提示框、询问框三种常用对话框，还可以自己实例化新的弹窗对象，并轻松自定义新的皮肤样式。

使用说明

1. 弹出气泡，并在两秒后自动消失


    FreeDialog.tips('我马上会自动消失的。', 2000);

2. 弹出提示框，点击确定后消失


    FreeDialog.alert(['这是一条消息。', '点击确定才会消失。']);

3. 弹出提示框，点击确定后消失，点取消后不消失


    FreeDialog.ask('你确定要关闭这个对话框吗？',
        function(){ // 点击确定键的回调函数
            FreeDialog.tips('关闭了。');
        },
        function(){ // 点击取消键的回调函数
            FreeDialog.tips('关闭被阻止。');
            return false;
    // 返回false可阻止默认关闭行为
        }
    );

4. 修改提示框皮肤（会影响后续提示框效果）


    FreeDialog.alertDialog.setSkin('skin-orange');
    FreeDialog.alert('提示框变为黄色皮肤。');

5. 修改询问框标题（不会影响后续询问框效果）


    FreeDialog.askDialog.setTitle('用户调查');
    FreeDialog.ask('请问您想要参与我们的用户调查吗？',
        function(){
            FreeDialog.tips('好的，谢谢您的参与！');
        },
        function(){
            FreeDialog.tips('真是遗憾，那下次再会吧！');
        }
    );

6. 实例化新的对话框对象


    if (!window.myDialog)
    {
        window.myDialog = new
        FreeDialog({
            title: '自定义对话框',
            content: [
                '这是一个自定义的对话框，点击周围可以关闭。',
                document.getElementById('tagContent')
            ],
            clickAroundClose: true, // 点击对话框周围可关闭该对话框
            skin: 'skin-orange' // 默认皮肤
        });
    }
    window.myDialog.show();
